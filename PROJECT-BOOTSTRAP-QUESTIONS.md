# Project Bootstrap Questions

Use this on the first meaningful `@dev-expert` call in a new workspace.

Ask at most 6 grouped questions.
Keep them short and practical.

## Ask about
1. Stack + runtime
2. Install / dev / build / test / lint commands
3. Important folders, entrypoints, and architecture rules
4. Coding, naming, docs, and testing preferences
5. Security, deployment, and "ask first" constraints
6. Definition of done

## Goal
Fill `PROJECT-WORKSPACE-MEMORY.md` with confirmed context so the agent can support the project better on later calls.
